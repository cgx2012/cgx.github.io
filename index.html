hexo 配置遇到的问题

Posted on 2018-10-06 | In Environment Configuration |

整了一天才把大体的弄好，这里记录遇到了两个卡了比较久的问题，其他的配置网上的教程其实挺多的

  1. github page 404
    在设置好Repository 后，注意repository 的名称是github名称+.github.io。直接访问对应的域名。总是报404，后面在repository的设置里面设置了一下主题解决了

  2. 设置 hexo 的 tags和category的时候，打开页面总是 Cannot GET /tags/ or categories 参照网上的解决方案对了很久，配置没有啥问题，后面留意到iterm有error信息，(内容比较多被刷上去了，同时还是能正常genter generate,一开始没注意到)
    error 信息是

    ERROR Process failed: tags/index.md
    TypeError: Cannot read property ‘utcOffset’ of null

解决方法是：将hexo _config.yml 和 主题的 _config.yml 时区设置成一样的
